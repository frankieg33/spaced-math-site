# 每日數學 — 支援

常見問題的快速解答，以及如何聯絡我們。

## 開始使用

1. 打開每日數學。
2. 在歡迎畫面上，輕點**建立檔案**。選擇名稱、表情符號和顏色。
3. 在檔案主畫面上，輕點某一堆（`+ − × ÷`）或**全部運算**，開始一次學習。說出的答案和輸入的答案都可以。

如果你在另一部裝置上開啟了 iCloud 同步並使用過每日數學，請改為在歡迎畫面上輕點**從 iCloud 還原**。iCloud 同步完成後（通常幾秒內），你的檔案就會出現。如果你在另一部裝置上加入了某個家庭，你會看到一張確認卡片，提示你在這裡也加入。

## 免費版與完整版

每日數學免費提供一個檔案和一種運算（你首次打開 App 時選擇的那一種）。一次性購買的完整解鎖會新增：

- 全部四種運算以及混合練習
- 最多 8 個檔案，各自獨立的進度
- 在家人裝置之間進行家人共享
- 自訂練習和計時挑戰

這是一次性購買，不是訂閱，並透過家人共享與你的家人共享。

**如何解鎖完整版？**
打開**設定 → 每日數學 完整版 → 解鎖完整版**，或輕點任何已鎖定的功能。先由大人解出一道簡短的乘法題（這是 Apple 對兒童類 App 的要求），接著由 App Store 處理付款。

**我已經付過款了。如何回復？**
在新裝置上或重新安裝後，用購買時所用的 Apple ID 登入 App Store，然後打開**設定 → 每日數學 完整版 → 回復購買項目**。如果是家人購買的，只要你們在同一個家人群組中，家人共享就會自動解鎖。

**我有促銷碼或禮品碼。**
打開**設定 → 每日數學 完整版 → 兌換代碼**並輸入。代碼被接受後，完整版會立即解鎖。

## 語音模式

預設情況下，App 會聆聽說出的答案。數字可以自然地說（「二十四」「二 四」「一百四十四」），也可以按數字說。

**語音模式無法運作，該怎麼辦？**

- 確認你已允許存取麥克風和語音辨識。iOS 設定 → 每日數學 → 麥克風（開）和語音辨識（開）。
- 輕點麥克風區域可放棄目前這次嘗試並重試。
- 如果辨識一直聽錯，請切換到螢幕上的鍵盤：在答案區域輕點**鍵盤**，或打開檔案設定 → 關閉**語音模式**。

**我說的是正確的數字，答案卻被判為「錯誤」。**
語音辨識可能會把你說的數字辨識成另一個數字。如果經常發生，請暫時切換到鍵盤，讓辨識逐漸適應（在同一部裝置上通常會隨時間改善）。

## 間隔重複

每個檔案各自保存獨立的進度。你從未見過的卡片是「新卡」，會立即到期。隨著你學習：

- 答對會把卡片推向更遠的將來。回答越快（預設低於 2.0 秒）間隔越長。
- 答錯會立刻把卡片帶回輪替：它會在同一次學習中往後 6–12 張重新排隊。
- 如果你在本次學習後面又把這張卡片做對了，只計入這次找回的評分，你的「錯誤」會被忽略。

檔案主畫面上的**統計**列顯示你的連續天數、每天的複習總數，以及已學會／總卡片數。

**複習**卡片（僅在有內容需要複習時顯示）是一個有針對性的訂正佇列：它會輪流呈現你最近做錯的卡片、處於重新學習中的卡片，以及前一天遺留到期的卡片。

## 練習（依檔案）

檔案主畫面上的**練習**列是一種沒有排程器的自由練習。卡片池相同，但會忽略到期日期。適合熱身，或適合你不想影響 FSRS 排程器的學習。練習難度（簡單／中等／困難）可在設定中設定。

## 啟用的數字

在設定 → 啟用的數字中，選擇你想針對哪些運算數（1–12）進行練習。該選擇會作為**基準**：選取 `{1, 2}` 時，你會看到涉及 1 或 2 的每一道題，而不只是 `1+2` / `2+2` / `1+1`。關閉再重新開啟某個數字，都不會失去它的 FSRS 狀態。

## iCloud 同步

**如何開啟？**
在檔案選擇器中輕點齒輪圖像 → **設定 → 同步 → 使用 iCloud → 開**。確認你已在 iOS 設定中登入 iCloud。

**我的檔案沒有同步。**

- 確認你在兩部裝置上都用*同一個* Apple ID 登入了 iCloud。
- 確認兩部裝置的 App 設定中同步都已開啟。
- 跨裝置的 CloudKit 同步通常很快（幾秒，常常透過靜默推播幾乎即時完成），但在網路較弱時可能更久。在每部裝置上打開 App，給它 30 秒來穩定下來。

**我切換了 Apple ID，現在檔案不見了。**
為了你的安全，App 會在偵測到裝置上的 iCloud 帳號發生變化時自動關閉同步，以免舊資料被悄悄上傳到另一個帳號。如果你的檔案仍在這部裝置上，它們安然無恙。如果它們只在之前的 iCloud 帳號裡，請切換回那個帳號並重新開啟同步即可還原。

**我想清除 iCloud 上的副本。**
打開管理檔案，逐個刪除每個檔案。開啟 iCloud 同步時，刪除也會從家庭區域移除對應的 CloudKit 記錄。

## 家庭

每日數學支援把你的整個家庭與其他 Apple ID 共享，這樣一個使用爸爸 iPad 和媽媽 iPhone 的孩子，即使這些裝置登入的是不同的 iCloud 帳號，也能從上次停下的地方接著來。在 v3 中，家庭是共享的單位：擁有者裝置上的每個檔案都屬於這個家庭。

**如何建立家庭？**

1. 確認 iCloud 同步已開啟（齒輪圖像 → 設定 → 同步 → 使用 iCloud → 開）。
2. 打開**設定 → 家庭 → 設定家庭**。
3. 為你的家庭取個名字（例如「陳家」）。
4. 選擇邀請方式：訊息、郵件、拷貝連結或 AirDrop。

**有人接受邀請後會怎樣？**

參與者輕點邀請連結，App 會帶著確認頁打開，他們可在以下之間選擇：

- **先匯出**：把目前的檔案儲存為一個 JSON 檔案（如果改變主意，之後可重新匯入）。
- **加入**：用家庭的檔案取代其目前的檔案。**這在參與者的裝置上無法復原** —— 他們加入前的檔案會從裝置上抹除（但不會從已經持有副本的 iCloud 帳號中抹除）。
- **取消**：什麼都不做。使用者之後可以再次輕點邀請連結。

加入後，參與者會在其選擇器中看到家庭名稱作為標籤，並可檢視和編輯家庭中的任何檔案。其學習活動會透過 CloudKit 的權限模型寫入擁有者的 iCloud。

**誰能看到這個家庭？**

只有你明確邀請的 Apple ID。共享僅限受邀者：任何拿到連結的人都可加入，但前提是你把連結傳給了他們。你可以隨時撤銷存取權限（設定 → 家庭 → 解散）。

**資料儲存在誰的 iCloud 中？**

擁有者的（建立家庭的人）。任何參與者記錄的學習活動都會透過 CloudKit 的權限模型寫入擁有者的 iCloud。參與者的 iCloud 不會保存單獨的副本。

**如果擁有者解散家庭或登出 iCloud 會怎樣？**

所有參與者會在下次重新整理時失去即時同步，但家庭檔案會**作為他們自己的檔案保留在裝置上** —— 不會被抹除。如果擁有者只是登出了 iCloud，重新登入即可為所有人還原即時同步。

**參與者能否在不牽涉擁有者的情況下離開家庭？**

可以。**設定 → 家庭 → 退出家庭**。擁有者的資料不受影響；此後參與者會把家庭的檔案作為自己的保留在裝置上。

**一個家庭最多可有多少參與者？**

Apple 的 `CKShare` 上限是 100。自然的家庭規模約為 6（與 Apple 家人共享群組的上限一致）。

**持有受管理 Apple ID 的 13 歲以下兒童能加入家庭嗎？**

受管理（學校發放）的 Apple ID 不能作為 CKShare 參與者。對於這類家庭，請用*同一個* Apple ID 登入兩部裝置：依 Apple ID 的同步路徑無需邀請也能運作。

## 匯出 / 匯入（備份）

在**設定 → 備份**中，你可以：

- **將檔案匯出為檔**：把目前裝置上的每個檔案儲存為一個 JSON 檔案。可用作 iCloud 之外的手動備份，或在任何無法復原的操作（加入家庭、更換裝置等）之前使用。
- **從檔案匯入檔案**：還原之前的匯出。**合併**模式會新增缺少的檔案，並在匯入副本的 `updatedAt` 較新時更新現有檔案。**取代**模式會抹除裝置上的檔案並安裝匯入的那一組；僅在你目前不在某個家庭中時可用。

## 我想重新開始一個檔案而不刪除它

打開該檔案，輕點齒輪圖像，然後選擇重設此檔案的進度。這會清除間隔重複狀態、錯誤和活動記錄，但保留檔案的名稱、表情符號、顏色和設定。

## 我的裝置

- 每日數學可在搭載 iOS 16.0 或以上版本的 iPhone 和 iPad 上運作。
- 除 iCloud 同步（需要登入 iCloud）外，所有功能均可離線使用。

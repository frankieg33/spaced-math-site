# 隱私權政策 — 每日數學

**生效日期：** 2026-05-21
**最近更新：** 2026-06-10

本政策說明 iOS App 每日數學（「本 App」）如何處理你的資訊。它自 v1.0 起適用，包括取代 v2.0 依檔案共享的 v3 單一家庭 CloudKit 共享模型。

## 簡明摘要

- 我們不營運任何伺服器。我們不收集任何個人資訊。
- 除非你開啟 **iCloud 同步**，否則一切都留在你的裝置上；開啟後，由 Apple（而非我們）將其儲存在你的私人 iCloud 帳號中。
- 本 App 僅在你於語音模式下作答時使用麥克風。音訊由 Apple 的裝置端（Siri 等級）語音辨識處理並立即捨棄；我們絕不儲存、上傳或分析它。
- 沒有任何第三方追蹤器、廣告或分析 SDK。

## 本 App 在你裝置上儲存的內容

- **檔案。** 名稱、表情符號、顏色，以及依檔案的設定。
- **間隔重複狀態。** 對每張卡片（例如 `7 + 8`）：排定的下次複習日期、難度、穩定度和複習次數。
- **活動。** 依檔案的每日已完成複習計數（用於連續天數標籤和統計畫面）。
- **錯誤。** 你最近答錯的題目佇列，以便本 App 重新呈現它們。
- **偏好設定。** App 範圍的開關，例如是否開啟 iCloud 同步。

所有這些都寫入本 App 自己的沙盒化「文件」檔案夾和 `UserDefaults`。本 App 無法讀取其他 App 的資料，其他 App 也無法讀取本 App 的資料。

## 本 App 選擇性地在 iCloud 中儲存的內容

iCloud 同步**預設關閉**。當你開啟它（齒輪圖像 → **設定 → 同步 → 使用 iCloud**）時，本 App 會透過 Apple 的 **CloudKit** API 把上述相同資料寫入你的私人 iCloud。這是你的 iCloud，只有你在使用同一 Apple ID 登入的裝置上才能存取。我們無權存取。

- 具體同步的資料：檔案、卡片狀態、活動、錯誤，以及（選擇性的）家人共享中繼資料（你為家庭選擇的一個顯示名稱）。
- iCloud 同步**不包含**任何語音錄音的內容，也不包含任何分析。
- 當你用不同的 Apple ID 登入時，本 App 會偵測到變化並關閉同步，直到你明確重新開啟，以免資料被悄悄上傳到另一個帳號。
- 在**管理檔案**中刪除某個檔案，會將該檔案從裝置移除，並（當 iCloud 在原始 Apple ID 上可連線時）一併從你的 iCloud 副本中移除。

## 家人共享

在 v3 中，共享發生在**家庭**層級，而非依檔案。打開齒輪圖像 → **設定 → 家庭 → 設定家庭**，為你的家庭命名，並透過標準的 iOS 共享選單（訊息、郵件或 AirDrop）邀請家人。

- 擁有者裝置上的所有檔案都屬於這個家庭。邀請一位家人會把它們一次性全部共享。
- 資料位於**家庭擁有者的 iCloud** 中。參與者不會在自己的 iCloud 中保存單獨副本；其學習活動會透過 Apple 依區域劃分的 CKShare 權限模型直接寫入擁有者的 CloudKit。
- 加入家庭在**加入的裝置上無法復原**：參與者現有的本機檔案會被家庭的檔案取代。加入前，本 App 會提供**將檔案匯出為檔**選項，以便你保存目前資料的 JSON 副本，若改變主意可在之後重新匯入。
- 如果擁有者解散家庭（設定 → 家庭 → 解散）或登出 iCloud，參與者會在下次同步時失去即時存取。家庭檔案的本機快取會成為他們自己的（離開時不會進行無法復原的抹除）。
- 參與者可隨時離開家庭（設定 → 家庭 → 退出）。離開後，家庭的檔案會作為他們自己的保留在裝置上。
- Apple 的 `CKShare` 上限是每個共享項目 100 名參與者。自然的家庭規模約為 6。

## 匯出 / 匯入（JSON 備份）

在**設定 → 備份**中，你可以：

- **將檔案匯出為檔**：儲存一個包含目前裝置上所有檔案（無論家庭狀態）的 JSON 檔案。可用作手動備份或在無法復原的操作之前使用。
- **從檔案匯入檔案**：還原之前的匯出。**合併**模式會新增缺少的檔案，並在匯入副本較新時更新現有檔案。**取代**模式會抹除本機檔案並安裝匯入的那一組；僅在你不在某個家庭中時可用。

## 麥克風與語音辨識

當語音模式開啟時，在你作答期間，本 App 會：

1. 使用 `AVAudioEngine` 擷取即時麥克風音訊。
2. 將其串流到 `SFSpeechRecognizer`（Apple 的框架）。在較新的 iOS 裝置上，辨識預設在裝置端執行；某些設定可能使用 Apple 的語音辨識伺服器。
3. 讀取轉錄出的數字並評分答案。
4. 一旦取得最終答案就停止擷取。

我們不保留任何音訊、轉錄文字或衍生訊號。除 Apple 的 `SFSpeechRecognizer` 外，我們不向任何一方傳輸音訊。本 App 在首次使用時請求 `NSMicrophoneUsageDescription` 和 `NSSpeechRecognitionUsageDescription`；若任一權限被拒絕，語音模式會自動停用，改用螢幕上的數字鍵盤。

## 診斷記錄

本 App 透過 Apple 的標準 `OSLog` 框架輸出非結構化的診斷事件（例如「學習結束，正確=12 錯誤=3」）。這些記錄：

- 留在裝置上。
- 僅在裝置連接到 Mac 時，你才能在 Console.app 中看到。
- 對任何可能識別身分的值（檔案 ID、檔案名稱）使用 Apple 的 `.private` 隱私權分類，因此在發佈版本中這些值會被隱去。

我們不收集、擷取或傳輸 `OSLog` 內容。

## 我們*不*收集的資料

- 你的姓名、電子郵件或聯絡方式。
- 你的位置。
- 任何裝置識別碼（IDFA、IDFV、廣告 ID）。
- 超出 Apple 可能透過你的 iOS 設定收集範圍的當機報告。
- 任何分析或行為遙測。
- 付款或帳單資訊（購買完全由 Apple 處理）。

## 兒童隱私

每日數學旨在對兒童安全。由於本 App 不收集個人資訊且不聯絡第三方，它符合 COPPA 的「無個人資訊」定義。我們不顯示廣告，也沒有帳號或社交功能。每日數學包含一個一次性的 App 內購買（完整版的解鎖）；由於本 App 為兒童設計，依照 Apple 兒童類別的規則，任何購買之前都會顯示成人驗證。所有付款和任何代碼兌換都由 Apple 透過 App Store 處理。本 App 絕不會看到或儲存你的付款資訊。

## 你的選擇

- **關閉語音模式**：隨時依檔案關閉（設定 → 語音模式），或在 iOS 設定中拒絕麥克風權限以全域關閉。
- **關閉 iCloud 同步**：隨時（齒輪圖像 → 設定 → 同步 → 使用 iCloud → 關）。關閉不會刪除 iCloud 副本；若要移除，請逐個刪除每個檔案。
- **解散或離開家庭**（設定 → 家庭 → 解散 / 退出）以停止共享。解散會撤銷參與者對你家庭的存取；離開會把家庭的檔案作為你自己的保留在裝置上。
- **將檔案匯出為檔**（設定 → 備份 → 匯出）並保存在本機：一份絕不接觸 iCloud 的手動備份。
- **重設檔案的進度**而不刪除該檔案（檔案設定 → 重設）。
- **刪除檔案**（管理檔案 → 滑動或輕點刪除）：開啟同步時也會移除 iCloud 副本。逐個檔案重複以移除全部。

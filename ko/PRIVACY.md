# 개인정보 처리방침 — 매일 수학

**발효일:** 2026-05-21
**최종 업데이트:** 2026-06-10

이 방침은 iOS 앱 매일 수학("본 앱")이 사용자의 정보를 어떻게 처리하는지 설명합니다. v1.0 이후에 적용되며, v2.0의 프로필별 공유를 대체하는 v3의 단일 가족 CloudKit 공유 모델을 포함합니다.

## 쉬운 말 요약

- 우리는 어떤 서버도 운영하지 않습니다. 어떤 개인정보도 수집하지 않습니다.
- **iCloud 동기화**를 켜지 않는 한 모든 것은 사용자의 기기에 남습니다. 켠 경우에는 우리가 아니라 Apple이 사용자의 비공개 iCloud 계정에 저장합니다.
- 본 앱은 음성 모드에서 문제에 답하는 동안에만 마이크를 사용합니다. 오디오는 Apple의 기기 내(Siri 수준) 음성 인식으로 처리되어 즉시 폐기됩니다. 우리는 절대 저장, 업로드, 분석하지 않습니다.
- 제3자 추적기, 광고, 분석 SDK가 전혀 없습니다.

## 본 앱이 기기에 저장하는 것

- **프로필.** 이름, 이모지, 색상, 프로필별 설정.
- **간격 반복 상태.** 각 카드(예: `7 + 8`)에 대해 예정된 다음 복습 날짜, 난이도, 안정도, 복습 횟수.
- **활동.** 프로필별 하루 완료 복습 수(연속 기록 칩과 통계 화면에 사용).
- **실수.** 최근에 틀린 문제의 대기열로, 본 앱이 다시 표시할 수 있도록 합니다.
- **환경설정.** iCloud 동기화 사용 여부 같은 앱 전체 스위치.

이 모든 것은 본 앱 전용의 샌드박스화된 도큐멘트 폴더와 `UserDefaults`에 기록됩니다. 본 앱은 다른 앱의 데이터를 읽을 수 없고, 다른 앱도 본 앱의 데이터를 읽을 수 없습니다.

## 본 앱이 선택적으로 iCloud에 저장하는 것

iCloud 동기화는 **기본적으로 꺼져 있습니다.** 켜면(톱니바퀴 아이콘 → **설정 → 동기화 → iCloud 사용**), 본 앱은 위에서 설명한 동일한 데이터를 Apple의 **CloudKit** API를 통해 사용자의 비공개 iCloud에 기록합니다. 이는 사용자의 iCloud이며, 같은 Apple ID로 로그인한 기기에서 사용자만 접근할 수 있습니다. 우리는 접근할 수 없습니다.

- 동기화되는 정확한 데이터: 프로필, 카드 상태, 활동, 실수, 그리고 (선택적으로) 가족 공유 메타데이터(사용자가 가족에 정한 표시 이름).
- iCloud 동기화에는 음성 녹음 내용이나 어떤 분석도 **포함되지 않습니다.**
- 다른 Apple ID로 로그인하면 본 앱이 변경을 감지하고, 사용자가 명시적으로 다시 켤 때까지 동기화를 끕니다. 데이터가 다른 계정으로 조용히 업로드되지 않도록 하기 위해서입니다.
- **프로필 관리**에서 프로필을 삭제하면 그 프로필이 기기에서 제거되고, (원래 Apple ID에서 iCloud에 연결 가능한 경우) iCloud 사본에서도 제거됩니다.

## 가족 공유

v3에서는 공유가 프로필별이 아니라 **가족** 수준에서 이루어집니다. 톱니바퀴 아이콘 → **설정 → 가족 → 가족 설정**을 열고 가족 이름을 정한 뒤, 표준 iOS 공유 시트(메시지, Mail 또는 AirDrop)로 가족 구성원을 초대합니다.

- 소유자 기기의 모든 프로필이 가족에 속합니다. 가족 구성원을 초대하면 그것들이 한 번에 모두 공유됩니다.
- 데이터는 **가족 소유자의 iCloud**에 있습니다. 참가자는 자신의 iCloud에 별도 사본을 저장하지 않습니다. 학습 활동은 Apple의 영역 범위 CKShare 권한 모델을 통해 소유자의 CloudKit에 직접 기록됩니다.
- 가족 참여는 **참여하는 기기에서 되돌릴 수 없습니다.** 참가자의 기존 로컬 프로필이 가족의 프로필로 대체됩니다. 참여 전에 본 앱은 **프로필을 파일로 내보내기** 옵션을 제공하므로, 현재 데이터의 JSON 사본을 저장하고 마음이 바뀌면 나중에 다시 가져올 수 있습니다.
- 소유자가 가족을 해산하거나(설정 → 가족 → 해산) iCloud에서 로그아웃하면, 참가자는 다음 새로 고침에서 실시간 접근을 잃습니다. 가족 프로필의 로컬 캐시는 그들 자신의 것이 됩니다(나갈 때 되돌릴 수 없는 삭제는 없습니다).
- 참가자는 언제든지 가족을 나갈 수 있습니다(설정 → 가족 → 나가기). 나가도 가족의 프로필은 자신의 것으로 기기에 남습니다.
- Apple의 `CKShare` 한도는 공유 항목당 100명입니다. 자연스러운 가족 규모는 약 6명입니다.

## 내보내기 / 가져오기(JSON 백업)

**설정 → 백업**에서 다음을 할 수 있습니다.

- **프로필을 파일로 내보내기**: 현재 기기의 모든 프로필(가족 상태와 무관)을 담은 JSON 파일을 저장합니다. 수동 백업이나 되돌릴 수 없는 작업 전에 유용합니다.
- **파일에서 프로필 가져오기**: 이전 내보내기를 복원합니다. **병합** 모드는 누락된 프로필을 추가하고 가져온 사본이 더 최신이면 기존 프로필을 업데이트합니다. **대체** 모드는 로컬 프로필을 지우고 가져온 모음을 설치합니다(가족에 참여하지 않은 경우에만 사용 가능).

## 마이크와 음성 인식

음성 모드가 켜져 있을 때, 문제에 답하는 동안 본 앱은 다음을 합니다.

1. `AVAudioEngine`으로 실시간 마이크 오디오를 캡처합니다.
2. 이를 `SFSpeechRecognizer`(Apple의 프레임워크)로 스트리밍합니다. 최신 iOS 기기에서는 인식이 기본적으로 기기 내에서 실행됩니다. 일부 구성은 Apple의 음성 인식 서버를 사용할 수 있습니다.
3. 전사된 숫자를 읽고 답을 채점합니다.
4. 최종 답을 얻는 즉시 캡처를 중지합니다.

우리는 오디오, 전사본, 파생 신호를 전혀 보관하지 않습니다. Apple의 `SFSpeechRecognizer` 외의 어떤 곳에도 오디오를 전송하지 않습니다. 본 앱은 처음 사용할 때 `NSMicrophoneUsageDescription`과 `NSSpeechRecognitionUsageDescription`을 요청합니다. 둘 중 하나라도 거부되면 음성 모드가 자동으로 비활성화되고 화면의 숫자 키패드가 사용됩니다.

## 진단 로그

본 앱은 Apple의 표준 `OSLog` 프레임워크를 통해 비정형 진단 이벤트를 내보냅니다(예: "학습 세션 종료, 정답=12 오답=3"). 이 로그는:

- 기기에 남습니다.
- 기기가 Mac에 연결되어 있는 동안 Console.app에서 사용자만 볼 수 있습니다.
- 식별 가능성이 있는 값(프로필 ID, 프로필 이름)에는 Apple의 `.private` 개인정보 분류를 사용하므로, 출시 빌드에서는 그 값들이 가려집니다.

우리는 `OSLog` 내용을 수집, 검색, 전송하지 않습니다.

## 우리가 수집하지 *않는* 데이터

- 사용자의 이름, 이메일 또는 연락처.
- 사용자의 위치.
- 모든 기기 식별자(IDFA, IDFV, 광고 ID).
- 사용자의 iOS 설정을 통해 Apple이 수집할 수 있는 범위를 넘는 충돌 보고서.
- 모든 분석 또는 행동 텔레메트리.
- 결제 또는 청구 정보(구입은 전적으로 Apple이 처리합니다).

## 아동 개인정보

매일 수학은 어린이에게 안전하도록 설계되었습니다. 본 앱은 개인정보를 수집하지 않고 제3자에 연결하지 않으므로 COPPA의 "개인정보 없음" 정의를 충족합니다. 광고를 표시하지 않으며 계정이나 소셜 기능이 없습니다. 매일 수학에는 한 번만 하는 앱 내 구입(전체 앱 잠금 해제)이 하나 있습니다. 앱이 어린이를 위해 설계되었으므로, Apple 키즈 카테고리 규칙이 요구하는 대로 모든 구입 전에 성인 확인 절차가 표시됩니다. 모든 결제와 코드 사용은 Apple이 App Store를 통해 처리합니다. 본 앱은 사용자의 결제 정보를 절대 보거나 저장하지 않습니다.

## 사용자의 선택

- **음성 모드 끄기**: 언제든지 프로필별로(설정 → 음성 모드), 또는 iOS 설정에서 마이크 권한을 거부해 전체적으로.
- **iCloud 동기화 끄기**: 언제든지(톱니바퀴 아이콘 → 설정 → 동기화 → iCloud 사용 → 끄기). 끈다고 iCloud 사본이 삭제되지는 않습니다. 제거하려면 각 프로필을 개별적으로 삭제하세요.
- **가족 해산 또는 나가기**(설정 → 가족 → 해산 / 나가기)로 공유 중단. 해산하면 참가자의 가족 접근 권한이 취소됩니다. 나가면 가족의 프로필이 자신의 것으로 기기에 남습니다.
- **프로필을 파일로 내보내**(설정 → 백업 → 내보내기) 로컬에 보관: iCloud에 전혀 닿지 않는 수동 백업.
- **프로필의 진행 상황 재설정**: 프로필을 삭제하지 않고(프로필 설정 → 재설정).
- **프로필 삭제**(프로필 관리 → 쓸어넘기거나 탭해 삭제): 동기화가 켜져 있으면 iCloud 사본도 제거됩니다. 모두 제거하려면 프로필별로 반복하세요.

## 이 방침의 변경

이 방침을 변경하면 새 버전을 같은 URL에서 **최종 업데이트** 날짜를 갱신해 제공합니다. 중요한 변경은 본 앱의 출시 노트에도 안내합니다.

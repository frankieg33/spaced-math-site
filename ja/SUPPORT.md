# まいにち算数 — サポート

よくある質問への簡単な回答と、お問い合わせ方法です。

## はじめに

1. まいにち算数を開きます。
2. ようこそ画面で **プロフィールを作成** をタップします。名前、絵文字、色を選びます。
3. プロフィールのホームで、山（`+ − × ÷`）または **すべての演算** をタップして学習セッションを始めます。話した答えも入力した答えも使えます。

別のデバイスで iCloud 同期をオンにしてまいにち算数を使っていた場合は、ようこそ画面で **iCloud から復元** をタップしてください。iCloud の同期が完了すると（通常は数秒以内）、プロフィールが表示されます。別のデバイスでファミリーに参加している場合は、ここでも参加するよう促す確認カードが表示されます。

## 無料版とフル版

まいにち算数は、1つのプロフィールと1つの演算（アプリを初めて開いたときに選ぶ演算）で無料で使えます。一度きりの購入であるフルアンロックを行うと、次が追加されます。

- 4種類すべての演算と、混合の練習
- 最大8つのプロフィール（それぞれに個別の進捗）
- 家族のデバイス間でのファミリー共有
- カスタム練習とタイムトライアル

これはサブスクリプションではなく一度きりの購入で、ファミリー共有を通じてご家族と共有されます。

**フル版をアンロックするには？**
**設定 → まいにち算数 フル版 → フルアプリをアンロック** を開くか、ロックされた機能をタップします。まず大人が短い掛け算を解き（子ども向けアプリに対する Apple の要件です）、その後 App Store が支払いを処理します。

**すでに購入済みです。どう復元しますか？**
新しいデバイスや再インストール後は、購入時に使った Apple ID で App Store にサインインした状態で **設定 → まいにち算数 フル版 → 購入を復元** を開きます。ご家族が購入した場合は、同じファミリー共有グループにいれば、ファミリー共有で自動的にアンロックされます。

**プロモコードやギフトコードがあります。**
**設定 → まいにち算数 フル版 → コードを使う** を開いて入力します。コードが受け付けられるとすぐにフル版がアンロックされます。

## 音声モード

初期設定では、アプリは話した答えを聞き取ります。数は自然に（「にじゅうよん」「に よん」「ひゃくよんじゅうよん」）でも、数字としてでも言えます。

**音声モードが動きません。どうすれば？**

- マイクと音声認識へのアクセスを許可しているか確認してください。iOS の設定 → まいにち算数 → マイク（オン）と音声認識（オン）。
- マイクの領域をタップすると現在の入力を破棄して、やり直せます。
- 認識が繰り返し間違える場合は、画面上のキーパッドに切り替えます。答えの領域で **キーパッド** をタップするか、プロフィールの設定を開いて **音声モード** をオフにします。

**正しい数を言ったのに答えが「不正解」になります。**
音声認識は、言った数とは違う数を聞き取ることがあります。よく起きる場合は一時的にキーパッドに切り替え、認識が慣れるのを待ってください（同じデバイスでは時間とともに改善する傾向があります）。

## 間隔反復

プロフィールごとに進捗は独立しています。一度も見ていないカードは「新規」で、すぐに期限が来ます。学習を進めると：

- 正解するとカードはより先の未来へ送られます。速い解答（初期設定では 2.0 秒未満）ほど間隔が長くなります。
- 不正解のカードはすぐに巡回へ戻ります。同じセッションの 6〜12 枚あとに再び並びます。
- セッションの後半でそのカードを取り戻せた場合は、取り戻しの評価だけが反映され、「間違い」は忘れられます。

プロフィールのホームの **成績** バーには、連続記録、1日あたりの復習合計、習得済み/全カード数が表示されます。

**復習** カード（復習すべきものがあるときだけ表示）は、的を絞った修正キューです。最近間違えたカード、再学習中のカード、前の日から期限切れのカードを巡回します。

## 練習（プロフィールごと）

プロフィールのホームの **練習** バーは、スケジューラーなしの自由ドリルです。同じカードプールですが、期限を無視します。ウォームアップや、FSRS スケジューラーに影響を与えたくないセッションに便利です。練習の難易度（やさしい/ふつう/むずかしい）は設定で変更できます。

## 対象の数

設定 → 対象の数 で、どの被演算数（1〜12）を相手に練習するかを選びます。この選択は **基準** として働きます。`{1, 2}` を選ぶと、`1+2` / `2+2` / `1+1` だけでなく、1 または 2 を含むすべての問題が出ます。数をオフにして再びオンにしても、その FSRS の状態は失われません。

## iCloud 同期

**どう有効にしますか？**
プロフィール選択画面の歯車アイコンをタップ → **設定 → 同期 → iCloud を使う → オン**。iOS の設定で iCloud にサインインしていることを確認してください。

**プロフィールが同期されません。**

- 両方のデバイスで、*同じ* Apple ID で iCloud にサインインしているか確認してください。
- 両方のデバイスで、アプリの設定で同期がオンになっているか確認してください。
- デバイス間の CloudKit 同期は通常は高速です（数秒、サイレントプッシュでほぼ瞬時のことも多い）が、弱いネットワークではもっと時間がかかることがあります。各デバイスでアプリを開き、落ち着くまで 30 秒ほど待ってください。

**Apple ID を切り替えたらプロフィールが消えました。**
安全のため、アプリはデバイスの iCloud アカウントが変わったことを検知すると、自動的に同期をオフにします。古いデータが別のアカウントへ黙ってアップロードされないようにするためです。プロフィールがこのデバイスにまだ残っていれば、そのままです。以前の iCloud アカウントにしかなかった場合は、そのアカウントに戻して同期を再びオンにすると復元できます。

**iCloud のコピーを消したいです。**
プロフィールを管理 を開き、各プロフィールを個別に削除します。iCloud 同期がオンの場合、削除すると、対応する CloudKit のレコードもファミリーゾーンから削除されます。

## ファミリー

まいにち算数では、ファミリー全体を他の Apple ID と共有できます。パパの iPad とママの iPhone を使う子どもが、それらのデバイスが別々の iCloud アカウントにサインインしていても、やめたところからそのまま続けられます。v3 では、ファミリーが共有の単位です。所有者のデバイス上のすべてのプロフィールがファミリーに含まれます。

**ファミリーはどう作りますか？**

1. iCloud 同期がオンであることを確認します（歯車アイコン → 設定 → 同期 → iCloud を使う → オン）。
2. **設定 → ファミリー → ファミリーを設定** を開きます。
3. ファミリーに名前を付けます（例：「山田家」）。
4. 招待方法を選びます：メッセージ、メール、リンクをコピー、または AirDrop。

**招待を受け入れるとどうなりますか？**

参加者が招待リンクをタップすると、アプリが確認シートとともに開き、次から選びます。

- **先に書き出す**：現在のプロフィールを JSON ファイルに保存します（気が変わったら後で読み込み直せます）。
- **参加**：現在のプロフィールをファミリーのプロフィールに置き換えます。**これは参加者のデバイス上で取り消せません。** 参加前のプロフィールはデバイスから消えます（ただし、すでにコピーを持っている iCloud アカウントからは消えません）。
- **キャンセル**：何もしません。あとで招待リンクをもう一度タップできます。

参加後、参加者は選択画面にファミリー名のチップが表示され、ファミリー内のどのプロフィールも閲覧・編集できます。学習の記録は、CloudKit の権限モデルを通じて所有者の iCloud に書き込まれます。

**ファミリーは誰が見られますか？**

あなたが明示的に招待した Apple ID だけです。共有は招待制です。リンクを持っていれば誰でも参加できますが、それはあなたが送った相手に限られます。アクセスはいつでも取り消せます（設定 → ファミリー → 解散）。

**データは誰の iCloud に保存されますか？**

所有者（ファミリーを作成した人）の iCloud です。どの参加者が記録した学習活動も、CloudKit の権限モデルを通じて所有者の iCloud に書き込まれます。参加者の iCloud には別のコピーは保存されません。

**所有者がファミリーを解散したり iCloud からサインアウトしたら？**

すべての参加者は次回の更新でライブ同期を失いますが、ファミリーのプロフィールは **自分のものとしてデバイスに残ります。** 消えることはありません。所有者が iCloud からサインアウトしただけなら、再びサインインすれば全員のライブ同期が戻ります。

**参加者は所有者を介さずにファミリーから抜けられますか？**

はい。**設定 → ファミリー → ファミリーを退出**。所有者のデータには影響しません。参加者は以後、ファミリーのプロフィールを自分のものとしてデバイスに保持します。

**ファミリーには何人まで参加できますか？**

Apple の `CKShare` の上限は 100 です。自然なファミリーの規模は 6 程度です（Apple のファミリー共有グループの上限に対応します）。

**管理対象 Apple ID を持つ 13 歳未満の子どもはファミリーに参加できますか？**

管理対象（学校発行）の Apple ID は CKShare の参加者になれません。そうしたご家庭では、両方のデバイスを *同じ* Apple ID でサインインしてください。Apple ID ごとの同期経路なら、招待なしでも機能します。

## 書き出し / 読み込み（バックアップ）

**設定 → バックアップ** では次ができます。

- **プロフィールをファイルに書き出す**：現在のデバイス上のすべてのプロフィールを JSON ファイルとして保存します。iCloud 以外への手動バックアップや、取り消せない操作（ファミリーへの参加、デバイスの切り替えなど）の前に便利です。
- **プロフィールをファイルから読み込む**：以前の書き出しを復元します。**統合** モードは不足しているプロフィールを追加し、読み込んだコピーの `updatedAt` がより新しければ既存のものを更新します。**置き換え** モードはデバイスのプロフィールを消去して、読み込んだ一式をインストールします（ファミリーに参加していないときのみ利用できます）。

## プロフィールを削除せずに最初からやり直したい

プロフィールを開き、歯車アイコンをタップして、このプロフィールの進捗をリセット を選びます。間隔反復の状態、間違い、活動履歴は消えますが、プロフィールの名前、絵文字、色、設定は残ります。

## 私のデバイス

- まいにち算数は iOS 16.0 以降の iPhone と iPad で動作します。
- iCloud 同期（iCloud へのサインインが必要）を除き、すべての機能はオフラインで動作します。

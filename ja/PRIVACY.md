# プライバシーポリシー — まいにち算数

**発効日：** 2026-05-21
**最終更新日：** 2026-06-10

このポリシーは、iOS アプリ「まいにち算数」（以下「本アプリ」）がお客様の情報をどのように扱うかを説明します。v1.0 以降に適用され、v2.0 のプロフィール単位の共有に代わる v3 の単一ファミリー CloudKit 共有モデルも含みます。

## わかりやすい要約

- 私たちはサーバーを一切運用しません。個人情報を一切収集しません。
- **iCloud 同期** をオンにしない限り、すべてはお客様のデバイスに留まります。オンにした場合は、私たちではなく Apple が、お客様の非公開の iCloud アカウントに保存します。
- 本アプリは、音声モードで問題に答えている間だけマイクを使います。音声は Apple のデバイス内（Siri 相当）の音声認識で処理され、ただちに破棄されます。私たちが保存・アップロード・分析することは一切ありません。
- サードパーティのトラッカー、広告、分析 SDK は一切ありません。

## 本アプリがデバイスに保存するもの

- **プロフィール。** 名前、絵文字、色、プロフィールごとの設定。
- **間隔反復の状態。** 各カード（例：`7 + 8`）について、次回の復習予定日、難易度、安定度、復習回数。
- **活動。** プロフィールごとの 1 日あたりの完了した復習数（連続記録のチップと成績画面に使用）。
- **間違い。** 最近不正解だった問題のキュー。本アプリが再び出題できるようにするためのものです。
- **設定。** iCloud 同期が有効かどうかなど、アプリ全体のスイッチ。

これらはすべて、本アプリ専用のサンドボックス化された「書類」フォルダと `UserDefaults` に書き込まれます。本アプリは他のアプリのデータを読み取れず、他のアプリも本アプリのデータを読み取れません。

## 本アプリが任意で iCloud に保存するもの

iCloud 同期は **初期設定ではオフ** です。オンにすると（歯車アイコン → **設定 → 同期 → iCloud を使う**）、本アプリは上記と同じデータを、Apple の **CloudKit** API を通じてお客様の非公開の iCloud に書き込みます。これはお客様の iCloud であり、同じ Apple ID でサインインしたデバイスでお客様だけがアクセスできます。私たちはアクセスできません。

- 同期される具体的なデータ：プロフィール、カードの状態、活動、間違い、そして（任意で）ファミリー共有のメタデータ（お客様が選ぶファミリーの表示名）。
- iCloud 同期に音声録音の内容や分析は **含まれません**。
- 別の Apple ID でサインインすると、本アプリは変更を検知し、お客様が明示的に再びオンにするまで同期をオフにします。データが別のアカウントへ黙ってアップロードされないようにするためです。
- **プロフィールを管理** からプロフィールを削除すると、そのプロフィールはデバイスから削除され、（元の Apple ID で iCloud に到達できる場合は）iCloud のコピーからも削除されます。

## ファミリー共有

v3 では、共有はプロフィール単位ではなく **ファミリー** 単位で行われます。歯車アイコン → **設定 → ファミリー → ファミリーを設定** を開き、ファミリーに名前を付け、標準の iOS 共有シート（メッセージ、メール、AirDrop）で家族を招待します。

- 所有者のデバイス上のすべてのプロフィールがファミリーに属します。家族を招待すると、それらが一度にすべて共有されます。
- データは **ファミリーの所有者の iCloud** にあります。参加者は自分の iCloud に別のコピーを保存しません。学習活動は、Apple のゾーン単位の CKShare 権限モデルを通じて、所有者の CloudKit に直接書き込まれます。
- ファミリーへの参加は **参加するデバイス上で取り消せません。** 参加者の既存のローカルプロフィールはファミリーのものに置き換わります。参加前に、本アプリは **プロフィールをファイルに書き出す** 選択肢を提示するので、現在のデータの JSON コピーを保存し、気が変わったら後で読み込み直せます。
- 所有者がファミリーを解散したり（設定 → ファミリー → 解散）、iCloud からサインアウトしたりすると、参加者は次回の同期でライブアクセスを失います。ファミリーのプロフィールのローカルキャッシュは参加者自身のものになります（抜けるときに取り消せない削除は行われません）。
- 参加者はいつでもファミリーから抜けられます（設定 → ファミリー → 退出）。抜けても、ファミリーのプロフィールは自分のものとしてデバイスに残ります。
- Apple の `CKShare` の上限は共有項目あたり 100 人です。自然なファミリーの規模は約 6 人です。

## 書き出し / 読み込み（JSON バックアップ）

**設定 → バックアップ** では次ができます。

- **プロフィールをファイルに書き出す**：現在のデバイス上のすべてのプロフィール（ファミリーの状態を問わず）を含む JSON ファイルを保存します。手動バックアップや、取り消せない操作の前に便利です。
- **プロフィールをファイルから読み込む**：以前の書き出しを復元します。**統合** モードは不足しているプロフィールを追加し、読み込んだコピーがより新しければ既存のものを更新します。**置き換え** モードはローカルのプロフィールを消去して、読み込んだ一式をインストールします（ファミリーに参加していないときのみ利用できます）。

## マイクと音声認識

音声モードが有効なとき、問題に答えている間、本アプリは次を行います。

1. `AVAudioEngine` でマイクのライブ音声を取り込みます。
2. それを `SFSpeechRecognizer`（Apple のフレームワーク）にストリーミングします。最近の iOS デバイスでは認識は初期設定でデバイス内で実行されます。構成によっては Apple の音声認識サーバーを使うことがあります。
3. 書き起こされた数字を読み取り、答えを採点します。
4. 最終的な答えが得られ次第、取り込みを停止します。

私たちは音声、書き起こし、派生する信号を一切保持しません。Apple の `SFSpeechRecognizer` 以外のいかなる相手にも音声を送信しません。本アプリは初回使用時に `NSMicrophoneUsageDescription` と `NSSpeechRecognitionUsageDescription` を要求します。どちらかの許可が拒否された場合、音声モードは自動的に無効になり、画面上の数字キーパッドが使われます。

## 診断ログ

本アプリは、Apple の標準の `OSLog` フレームワークを通じて、非構造の診断イベントを出力します（例：「学習セッション終了、正解=12 不正解=3」）。これらのログは：

- デバイスに留まります。
- デバイスが Mac に接続されている間、Console.app でお客様だけが見られます。
- 識別につながりうる値（プロフィール ID、プロフィール名）には Apple の `.private` プライバシー分類を使うため、出荷ビルドではそれらの値は伏せられます。

私たちは `OSLog` の内容を収集・取得・送信しません。

## 私たちが収集 *しない* データ

- お客様の名前、メール、連絡先。
- お客様の位置情報。
- いかなるデバイス識別子（IDFA、IDFV、広告 ID）も。
- お客様の iOS 設定を通じて Apple が収集しうる範囲を超えるクラッシュレポート。
- いかなる分析・行動テレメトリも。
- 支払い・請求情報（購入はすべて Apple が処理します）。

## 子どものプライバシー

まいにち算数は、子どもにとって安全であるように設計されています。本アプリは個人情報を収集せず、第三者に接続しないため、COPPA の「個人情報なし」の定義を満たします。広告は表示せず、アカウントやソーシャル機能もありません。まいにち算数には一度きりのアプリ内課金（フル版のアンロック）が 1 つあります。アプリは子ども向けに設計されているため、Apple のキッズカテゴリの規則が求めるとおり、いかなる購入の前にも大人向けのゲートが表示されます。すべての支払いとコードの引き換えは Apple が App Store を通じて処理します。本アプリがお客様の支払い情報を見たり保存したりすることは決してありません。

## お客様の選択肢

- **音声モードをオフにする**：プロフィールごとにいつでも（設定 → 音声モード）、または iOS 設定でマイクの許可を拒否して全体で。
- **iCloud 同期をオフにする**：いつでも（歯車アイコン → 設定 → 同期 → iCloud を使う → オフ）。オフにしても iCloud のコピーは削除されません。削除したい場合は各プロフィールを個別に削除してください。
- **ファミリーを解散または退出する**（設定 → ファミリー → 解散 / 退出）して共有をやめる。解散すると参加者のあなたのファミリーへのアクセスが取り消されます。退出すると、ファミリーのプロフィールは自分のものとしてデバイスに残ります。
- **プロフィールをファイルに書き出して**（設定 → バックアップ → 書き出す）ローカルに保管する。iCloud に一切触れない手動バックアップです。
- **プロフィールの進捗をリセットする**（プロフィールの設定 → リセット）。プロフィール自体は削除しません。
- **プロフィールを削除する**（プロフィールを管理 → スワイプまたはタップで削除）。同期がオンのときは iCloud のコピーも削除されます。すべて削除するにはプロフィールごとに繰り返します。

## このポリシーの変更

このポリシーを変更する場合、新しいバージョンを同じ URL で、**最終更新日** を更新して提供します。重要な変更は本アプリのリリースノートにも記載します。

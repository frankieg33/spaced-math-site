# Maths du jour — Assistance

Réponses rapides aux questions courantes et comment nous contacter.

## Premiers pas

1. Ouvrez Maths du jour.
2. Sur l'écran d'accueil, touchez **Créer un profil**. Choisissez un nom, un emoji et une couleur.
3. Depuis l'accueil du profil, touchez une pile (`+ − × ÷`) ou **Toutes les opérations** pour démarrer une session d'étude. Les réponses orales comme saisies fonctionnent.

Si vous avez utilisé Maths du jour sur un autre appareil avec la synchronisation iCloud activée, touchez plutôt **Restaurer depuis iCloud** sur l'écran d'accueil. Vos profils apparaîtront une fois iCloud synchronisé (généralement en quelques secondes). Si vous avez rejoint une famille sur un autre appareil, une carte de confirmation vous proposera de la rejoindre ici aussi.

## L'app gratuite et l'app complète

Maths du jour est gratuite avec un profil et une opération (celle que vous choisissez à la première ouverture de l'app). Un seul achat unique, le déblocage complet, ajoute :

- Les quatre opérations plus l'entraînement mixte
- Jusqu'à 8 profils, chacun avec sa propre progression
- Le partage de la famille entre les appareils de votre foyer
- L'entraînement personnalisé et les essais chronométrés

C'est un achat unique, pas un abonnement, et il est partagé avec votre famille via le Partage familial.

**Comment débloquer l'app complète ?**
Ouvrez **Réglages → Maths du jour Complet → Débloquer l'application complète**, ou touchez une fonctionnalité verrouillée. Un adulte résout d'abord une courte multiplication (une exigence d'Apple pour les apps destinées aux enfants), puis l'App Store gère le paiement.

**J'ai déjà payé. Comment restaurer mon achat ?**
Sur un nouvel appareil ou après une réinstallation, ouvrez **Réglages → Maths du jour Complet → Restaurer les achats**, connecté à l'App Store avec l'identifiant Apple utilisé pour l'achat. Si un membre de la famille l'a acheté, le Partage familial le débloque dès que vous êtes dans le même groupe familial.

**J'ai un code promotionnel ou cadeau.**
Ouvrez **Réglages → Maths du jour Complet → Utiliser un code** et saisissez-le. L'app complète se débloque dès que le code est accepté.

## Mode vocal

Par défaut, l'app écoute les réponses orales. Les nombres peuvent être dits naturellement (« vingt-quatre », « deux quatre », « cent quarante-quatre ») ou sous forme de chiffres.

**Le mode vocal ne fonctionne pas, que faire ?**

- Assurez-vous d'avoir autorisé l'accès au micro et à la reconnaissance vocale. Réglages iOS → Maths du jour → Micro (activé) et Reconnaissance vocale (activée).
- Touchez la zone du micro pour annuler la tentative en cours et réessayez.
- Si la reconnaissance se trompe systématiquement, passez au clavier à l'écran : touchez **Clavier** dans la zone de réponse, ou ouvrez les réglages du profil → désactivez le **Mode vocal**.

**Ma réponse est « fausse » alors que j'ai dit le bon nombre.**
La reconnaissance vocale peut interpréter un nombre différent de celui que vous avez dit. Si cela arrive souvent, passez temporairement au clavier et laissez la reconnaissance s'ajuster (elle a tendance à s'améliorer avec le temps sur le même appareil).

## Répétition espacée

Chaque profil conserve une progression indépendante. Les cartes jamais vues sont « nouvelles » et dues immédiatement. À mesure que vous étudiez :

- Une bonne réponse repousse la carte plus loin dans le futur. Les réponses plus rapides (moins de 2,0 s par défaut) obtiennent des intervalles plus longs.
- Une mauvaise réponse ramène la carte dans la rotation : elle est replacée 6 à 12 cartes plus loin dans la même session.
- Si vous retrouvez la carte plus tard dans la session, seule la note de récupération compte ; votre « erreur » est oubliée.

La barre **Statistiques** sur l'accueil du profil affiche votre série, le total de révisions par jour et les cartes apprises/totales.

La carte **Réviser** (visible uniquement quand il y a quelque chose à réviser) est une file de correction ciblée : elle parcourt les cartes récemment ratées, les cartes en réapprentissage et les cartes en retard d'un jour précédent.

## Entraînement (par profil)

La barre **Entraînement** sur l'accueil du profil est un exercice libre sans planificateur. Le même ensemble de cartes, mais les dates d'échéance sont ignorées. Utile pour s'échauffer ou pour les sessions où vous ne voulez pas influencer le planificateur FSRS. La difficulté de l'entraînement (Facile/Moyen/Difficile) se configure dans les Réglages.

## Nombres actifs

Dans Réglages → Nombres actifs, choisissez contre quels opérandes (1–12) vous voulez vous entraîner. La sélection agit comme une **base** : avec `{1, 2}` sélectionnés, vous verrez tous les problèmes touchant 1 ou 2, pas seulement `1+2` / `2+2` / `1+1`. Désactivez puis réactivez un nombre sans perdre son état FSRS.

## Synchronisation iCloud

**Comment l'activer ?**
Touchez l'icône d'engrenage dans le sélecteur de profils → **Réglages → Synchronisation → Utiliser iCloud → activé**. Assurez-vous d'être connecté à iCloud dans les Réglages iOS.

**Mes profils ne se synchronisent pas.**

- Vérifiez que vous êtes connecté à iCloud sur les deux appareils, avec le *même* identifiant Apple.
- Vérifiez que la synchronisation est activée dans les réglages de l'app sur les deux appareils.
- La synchronisation CloudKit entre appareils est généralement rapide (quelques secondes, souvent quasi instantanée via les notifications push silencieuses), mais peut prendre plus de temps sur des réseaux faibles. Ouvrez l'app sur chaque appareil et laissez-lui 30 secondes pour se stabiliser.

**J'ai changé d'identifiant Apple et mes profils ont disparu.**
Pour votre sécurité, l'app détecte le changement de compte iCloud sur votre appareil et désactive automatiquement la synchronisation, afin que les anciennes données ne soient pas envoyées en silence vers un autre compte. Si vos profils sont encore sur cet appareil, ils sont intacts. S'ils n'étaient que dans le compte iCloud précédent, revenez à ce compte et réactivez la synchronisation pour les récupérer.

**Je veux effacer les copies iCloud.**
Ouvrez Gérer les profils et supprimez chaque profil individuellement. Avec la synchronisation iCloud activée, les suppressions retirent aussi les enregistrements CloudKit correspondants de la zone familiale.

## Familles

Maths du jour permet de partager toute votre famille avec d'autres identifiants Apple, afin qu'un enfant utilisant l'iPad de papa et l'iPhone de maman puisse reprendre exactement où il s'était arrêté, même lorsque ces appareils sont connectés à des comptes iCloud différents. En v3, la famille est l'unité de partage : tous les profils de l'appareil du propriétaire font partie de la famille.

**Comment créer une famille ?**

1. Assurez-vous que la synchronisation iCloud est activée (icône d'engrenage → Réglages → Synchronisation → Utiliser iCloud → activé).
2. Ouvrez **Réglages → Famille → Configurer la famille**.
3. Donnez un nom à votre famille (par ex. « Famille Dupont »).
4. Choisissez comment inviter : Messages, Mail, Copier le lien ou AirDrop.

**Que se passe-t-il quand quelqu'un accepte une invitation ?**

Le participant touche le lien d'invitation, l'app s'ouvre avec une feuille de confirmation, et il choisit entre :

- **Exporter d'abord** : enregistre ses profils actuels dans un fichier JSON (il pourra les réimporter plus tard s'il change d'avis).
- **Rejoindre** : remplace ses profils actuels par ceux de la famille. **C'est destructif sur l'appareil du participant** : ses profils antérieurs sont effacés de l'appareil (mais pas d'un compte iCloud qui en aurait déjà une copie).
- **Annuler** : ne fait rien. L'utilisateur pourra retoucher le lien d'invitation plus tard.

Après avoir rejoint, le participant voit le nom de la famille sous forme d'étiquette dans son sélecteur et peut voir et modifier n'importe quel profil de la famille. Son activité d'étude est écrite dans l'iCloud du propriétaire via le modèle d'autorisations de CloudKit.

**Qui peut voir la famille ?**

Seuls les identifiants Apple que vous invitez explicitement. Le partage se fait uniquement sur invitation : toute personne disposant du lien peut rejoindre, mais seulement si vous le lui avez envoyé. Vous pouvez révoquer l'accès à tout moment (Réglages → Famille → Dissoudre).

**Dans l'iCloud de qui les données sont-elles stockées ?**

Dans celui du propriétaire (la personne qui a créé la famille). L'activité d'étude enregistrée par tout participant est écrite dans l'iCloud du propriétaire via le modèle d'autorisations de CloudKit. L'iCloud du participant ne stocke pas de copie distincte.

**Et si le propriétaire dissout la famille ou se déconnecte d'iCloud ?**

Tous les participants perdent la synchronisation en direct à leur prochaine actualisation, mais les profils de la famille **restent sur leur appareil comme les leurs** : ils ne sont pas effacés. Si le propriétaire s'est seulement déconnecté d'iCloud, une reconnexion rétablit la synchronisation en direct pour tout le monde.

**Un participant peut-il quitter une famille sans impliquer le propriétaire ?**

Oui. **Réglages → Famille → Quitter la famille**. Les données du propriétaire ne sont pas affectées ; le participant conserve désormais les profils de la famille sur son appareil comme les siens.

**Combien de participants une famille peut-elle compter ?**

La limite `CKShare` d'Apple est de 100. L'échelle familiale naturelle est d'environ 6 (correspond au maximum du groupe Partage familial d'Apple).

**Un enfant de moins de 13 ans avec un identifiant Apple géré peut-il rejoindre une famille ?**

Les identifiants Apple gérés (fournis par l'école) ne peuvent pas être participants à un CKShare. Pour ces familles, connectez les deux appareils avec le *même* identifiant Apple : la voie de synchronisation par identifiant Apple fonctionne sans invitations.

## Exporter / Importer (sauvegarde)

Dans **Réglages → Sauvegarde**, vous pouvez :

- **Exporter les profils dans un fichier** : enregistre tous les profils de l'appareil actuel sous forme de fichier JSON. Utile comme sauvegarde manuelle hors iCloud ou avant toute opération destructive (rejoindre une famille, changer d'appareil, etc.).
- **Importer les profils depuis un fichier** : restaure un export précédent. Le mode **Fusionner** ajoute les profils manquants et met à jour les existants si la copie importée a un `updatedAt` plus récent. Le mode **Remplacer** efface les profils de l'appareil et installe l'ensemble importé ; disponible uniquement lorsque vous n'êtes pas dans une famille.

## Je veux recommencer un profil sans le supprimer

Ouvrez le profil, touchez l'icône d'engrenage, puis Réinitialiser la progression de ce profil. Cela efface l'état de répétition espacée, les erreurs et l'historique d'activité, mais conserve le nom, l'emoji, la couleur et les réglages du profil.

## Mon appareil

- Maths du jour fonctionne sur iPhone et iPad avec iOS 16.0 ou ultérieur.
- Toutes les fonctionnalités fonctionnent hors ligne, sauf la synchronisation iCloud (qui nécessite une connexion à iCloud).

## Contact

E-mail **spacedmath@gmail.com**.

Si vous avez trouvé un bug, merci d'indiquer :

- Votre version d'iOS (Réglages → Général → Informations → Version d'iOS).
- Le modèle de votre appareil.
- Si le problème survient en mode vocal, en mode clavier ou les deux.
- Une brève description de ce que vous attendiez par rapport à ce qui s'est passé.

Une réponse arrive généralement sous quelques jours.

# นโยบายความเป็นส่วนตัว — คณิตทุกวัน

**วันที่มีผลบังคับใช้:** 2026-05-21
**อัปเดตล่าสุด:** 2026-06-10

นโยบายนี้อธิบายว่าแอป iOS คณิตทุกวัน ("แอป") จัดการข้อมูลของคุณอย่างไร มีผลตั้งแต่เวอร์ชัน v1.0 เป็นต้นไป รวมถึงโมเดลการแชร์ CloudKit แบบครอบครัวเดียวใน v3 (ซึ่งแทนที่การแชร์ต่อโปรไฟล์ใน v2.0)

## สรุปแบบเข้าใจง่าย

- เราไม่มีเซิร์ฟเวอร์ใด ๆ เราไม่เก็บข้อมูลส่วนบุคคลใด ๆ
- ทุกอย่างอยู่บนอุปกรณ์ของคุณ เว้นแต่คุณจะเปิด **การซิงค์ iCloud** ซึ่งในกรณีนั้น Apple ไม่ใช่เรา เป็นผู้จัดเก็บไว้ในบัญชี iCloud ส่วนตัวของคุณ
- แอปใช้ไมโครโฟนเฉพาะตอนที่คุณกำลังตอบโจทย์ในโหมดเสียงเท่านั้น เสียงถูกประมวลผลด้วยการรู้จำเสียงพูดของ Apple (บนอุปกรณ์ ระดับ Siri) และถูกทิ้งทันที เราไม่เคยจัดเก็บ อัปโหลด หรือวิเคราะห์มัน
- ไม่มีตัวติดตามของบุคคลที่สาม ไม่มีโฆษณา และไม่มี SDK การวิเคราะห์ใด ๆ

## สิ่งที่แอปจัดเก็บบนอุปกรณ์ของคุณ

- **โปรไฟล์** ชื่อ อิโมจิ สี และการตั้งค่าต่อโปรไฟล์
- **สถานะการทบทวนแบบเว้นช่วง** สำหรับการ์ดแต่ละใบ (เช่น `7 + 8`): วันที่ทบทวนถัดไปที่กำหนดไว้ ความยาก ความเสถียร และจำนวนครั้งที่ทบทวน
- **กิจกรรม** จำนวนการทบทวนที่เสร็จต่อวันต่อโปรไฟล์ (ใช้สำหรับชิปสตรีคและหน้าจอสถิติ)
- **ข้อผิดพลาด** คิวของโจทย์ที่คุณเพิ่งตอบผิด เพื่อให้แอปนำกลับมาแสดงอีก
- **การตั้งค่าที่ชอบ** สวิตช์ทั้งแอป เช่น การซิงค์ iCloud เปิดอยู่หรือไม่

ทั้งหมดนี้ถูกเขียนลงโฟลเดอร์เอกสารที่แยกออกมาเป็นของแอปเอง และลง `UserDefaults` แอปไม่สามารถอ่านข้อมูลของแอปอื่น และแอปอื่นไม่สามารถอ่านข้อมูลของแอปนี้

## สิ่งที่แอปจัดเก็บใน iCloud แบบเลือกได้

การซิงค์ iCloud **ปิดอยู่โดยค่าเริ่มต้น** เมื่อคุณเปิด (ไอคอนรูปเฟือง → **การตั้งค่า → ซิงค์ → ใช้ iCloud**) แอปจะเขียนข้อมูลเดียวกับที่อธิบายข้างต้นลงใน iCloud ส่วนตัวของคุณผ่าน **CloudKit** API ของ Apple นี่คือ iCloud ของคุณ ซึ่งเข้าถึงได้เฉพาะคุณบนอุปกรณ์ที่ลงชื่อเข้าใช้ด้วย Apple ID เดียวกัน เราไม่มีสิทธิ์เข้าถึงมัน

- ข้อมูลที่ซิงค์อย่างแน่ชัด: โปรไฟล์ สถานะการ์ด กิจกรรม ข้อผิดพลาด และ (เลือกได้) ข้อมูลเมตาการแชร์ในครอบครัว (ชื่อที่แสดงที่คุณเลือกให้ครอบครัวของคุณ)
- การซิงค์ iCloud **ไม่** รวมเนื้อหาของการบันทึกเสียงใด ๆ หรือการวิเคราะห์ใด ๆ
- เมื่อคุณลงชื่อเข้าใช้ด้วย Apple ID อื่น แอปจะตรวจพบการเปลี่ยนแปลงและปิดการซิงค์จนกว่าคุณจะเปิดอีกครั้งอย่างชัดเจน เพื่อไม่ให้ข้อมูลถูกอัปโหลดไปยังบัญชีอื่นอย่างเงียบ ๆ
- การลบโปรไฟล์จาก **จัดการโปรไฟล์** จะนำโปรไฟล์ออกจากอุปกรณ์ และ (เมื่อ iCloud เข้าถึงได้บน Apple ID เดิม) ออกจากสำเนา iCloud ของคุณด้วย

## การแชร์ในครอบครัว

ใน v3 การแชร์เกิดขึ้นที่ระดับ **ครอบครัว** แทนที่จะเป็นต่อโปรไฟล์ เปิดไอคอนรูปเฟือง → **การตั้งค่า → ครอบครัว → ตั้งค่าครอบครัว** ตั้งชื่อครอบครัวของคุณ และเชิญสมาชิกในครอบครัวผ่านแผ่นแชร์มาตรฐานของ iOS (ข้อความ, Mail หรือ AirDrop)

- ทุกโปรไฟล์บนอุปกรณ์ของเจ้าของเป็นของครอบครัว การเชิญสมาชิกหนึ่งคนจะแชร์ทั้งหมดในคราวเดียว
- ข้อมูลอยู่ใน **iCloud ของเจ้าของครอบครัว** ผู้เข้าร่วมไม่จัดเก็บสำเนาแยกต่างหากใน iCloud ของตน กิจกรรมการเรียนของพวกเขาถูกเขียนตรงไปยัง CloudKit ของเจ้าของผ่านโมเดลสิทธิ์ CKShare แบบจำกัดโซนของ Apple
- การเข้าร่วมครอบครัว **ย้อนกลับไม่ได้บนอุปกรณ์ที่เข้าร่วม**: โปรไฟล์ในเครื่องที่มีอยู่ของผู้เข้าร่วมจะถูกแทนที่ด้วยโปรไฟล์ของครอบครัว ก่อนเข้าร่วม แอปเสนอตัวเลือก **ส่งออกโปรไฟล์เป็นไฟล์** เพื่อให้คุณบันทึกสำเนา JSON ของข้อมูลปัจจุบัน และนำเข้าใหม่ภายหลังได้หากเปลี่ยนใจ
- หากเจ้าของยุบครอบครัว (การตั้งค่า → ครอบครัว → ยุบ) หรือออกจากระบบ iCloud ผู้เข้าร่วมจะสูญเสียการเข้าถึงแบบสดในการซิงค์ครั้งถัดไป แคชในเครื่องของโปรไฟล์ครอบครัวจะกลายเป็นของพวกเขาเอง (ไม่มีการลบที่ย้อนกลับไม่ได้ตอนออก)
- ผู้เข้าร่วมออกจากครอบครัวได้ทุกเมื่อ (การตั้งค่า → ครอบครัว → ออก) เมื่อออก เขาจะเก็บโปรไฟล์ของครอบครัวไว้บนอุปกรณ์ของตนในฐานะของตนเอง
- ขีดจำกัด `CKShare` ของ Apple คือ 100 ผู้เข้าร่วมต่อรายการที่แชร์ ขนาดครอบครัวตามธรรมชาติอยู่ที่ประมาณ 6

## ส่งออก / นำเข้า (สำรองข้อมูล JSON)

ใน **การตั้งค่า → สำรองข้อมูล** คุณสามารถ:

- **ส่งออกโปรไฟล์เป็นไฟล์**: บันทึกไฟล์ JSON ที่มีทุกโปรไฟล์บนอุปกรณ์ปัจจุบัน (ไม่ว่าสถานะครอบครัวจะเป็นอย่างไร) มีประโยชน์เป็นการสำรองข้อมูลด้วยตนเองหรือก่อนการกระทำที่ย้อนกลับไม่ได้
- **นำเข้าโปรไฟล์จากไฟล์**: กู้คืนการส่งออกครั้งก่อน โหมด **ผสาน** เพิ่มโปรไฟล์ที่ขาดหายและอัปเดตที่มีอยู่หากสำเนาที่นำเข้าใหม่กว่า โหมด **แทนที่** ลบโปรไฟล์ในเครื่องแล้วติดตั้งชุดที่นำเข้า มีให้ใช้เฉพาะเมื่อคุณไม่ได้อยู่ในครอบครัว

## ไมโครโฟนและการรู้จำเสียงพูด

เมื่อโหมดเสียงเปิดอยู่ ขณะที่คุณตอบโจทย์ แอปจะ:

1. จับเสียงไมโครโฟนแบบสดด้วย `AVAudioEngine`
2. สตรีมไปยัง `SFSpeechRecognizer` (เฟรมเวิร์กของ Apple) บนอุปกรณ์ iOS รุ่นใหม่ การรู้จำจะทำงานบนอุปกรณ์โดยค่าเริ่มต้น บางการตั้งค่าอาจใช้เซิร์ฟเวอร์การรู้จำเสียงพูดของ Apple
3. อ่านตัวเลขที่ถอดความแล้วและให้คะแนนคำตอบ
4. หยุดการจับทันทีที่ได้คำตอบสุดท้าย

เราไม่เก็บเสียง การถอดความ หรือสัญญาณที่ได้มาใด ๆ เราไม่ส่งเสียงไปยังฝ่ายใดนอกจาก `SFSpeechRecognizer` ของ Apple แอปขอ `NSMicrophoneUsageDescription` และ `NSSpeechRecognitionUsageDescription` เมื่อใช้ครั้งแรก หากสิทธิ์ใดสิทธิ์หนึ่งถูกปฏิเสธ โหมดเสียงจะถูกปิดโดยอัตโนมัติและใช้คีย์แพดตัวเลขบนหน้าจอแทน

## บันทึกการวินิจฉัย

แอปส่งเหตุการณ์การวินิจฉัยแบบไม่มีโครงสร้างผ่านเฟรมเวิร์ก `OSLog` มาตรฐานของ Apple (เช่น "จบเซสชันการเรียน, ถูก=12 ผิด=3") บันทึกเหล่านี้:

- อยู่บนอุปกรณ์
- มองเห็นได้เฉพาะคุณเท่านั้น ใน Console.app ขณะที่อุปกรณ์เชื่อมต่อกับ Mac
- ใช้การจัดประเภทความเป็นส่วนตัว `.private` ของ Apple สำหรับค่าใด ๆ ที่อาจระบุตัวตนได้ (รหัสโปรไฟล์ ชื่อโปรไฟล์) ดังนั้นในบิลด์ที่เผยแพร่ ค่าเหล่านั้นจะถูกปิดบัง

เราไม่เก็บ ดึง หรือส่งเนื้อหาของ `OSLog`

## ข้อมูลที่เราไม่เก็บ

- ชื่อ อีเมล หรือข้อมูลติดต่อของคุณ
- ตำแหน่งของคุณ
- ตัวระบุอุปกรณ์ใด ๆ (IDFA, IDFV, รหัสโฆษณา)
- รายงานข้อขัดข้องเกินกว่าที่ Apple อาจเก็บผ่านการตั้งค่า iOS ของคุณ
- การวิเคราะห์หรือเทเลเมทรีพฤติกรรมใด ๆ
- ข้อมูลการชำระเงินหรือการเรียกเก็บเงิน (Apple จัดการการซื้อทั้งหมด)

## ความเป็นส่วนตัวของเด็ก

คณิตทุกวันออกแบบมาให้ปลอดภัยสำหรับเด็ก เนื่องจากแอปไม่เก็บข้อมูลส่วนบุคคลและไม่ติดต่อบุคคลที่สาม จึงตรงตามนิยาม "ไม่มีข้อมูลส่วนบุคคล" ของ COPPA เราไม่แสดงโฆษณา และไม่มีบัญชีหรือฟีเจอร์โซเชียล คณิตทุกวันมีการซื้อในแอปครั้งเดียวหนึ่งรายการ (การปลดล็อกแอปแบบเต็ม) เนื่องจากแอปออกแบบมาสำหรับเด็ก จะมีการแสดงด่านสำหรับผู้ใหญ่ก่อนการซื้อใด ๆ ตามที่กฎหมวดเด็กของ Apple กำหนด การชำระเงินทั้งหมดและการแลกรหัสใด ๆ จัดการโดย Apple ผ่าน App Store แอปไม่เคยเห็นหรือจัดเก็บรายละเอียดการชำระเงินของคุณ

## ตัวเลือกของคุณ

- **ปิดโหมดเสียง** ได้ทุกเมื่อต่อโปรไฟล์ (การตั้งค่า → โหมดเสียง) หรือทั้งหมดโดยปฏิเสธสิทธิ์ไมโครโฟนในการตั้งค่า iOS
- **ปิดการซิงค์ iCloud** ได้ทุกเมื่อ (ไอคอนรูปเฟือง → การตั้งค่า → ซิงค์ → ใช้ iCloud → ปิด) การปิดไม่ลบสำเนา iCloud หากต้องการลบ ให้ลบแต่ละโปรไฟล์ทีละอัน
- **ยุบหรือออกจากครอบครัว** (การตั้งค่า → ครอบครัว → ยุบ / ออก) เพื่อหยุดการแชร์ การยุบจะเพิกถอนการเข้าถึงครอบครัวของคุณจากผู้เข้าร่วม การออกจะเก็บโปรไฟล์ของครอบครัวไว้บนอุปกรณ์ของคุณในฐานะของคุณเอง
- **ส่งออกโปรไฟล์เป็นไฟล์** (การตั้งค่า → สำรองข้อมูล → ส่งออก) และเก็บไว้ในเครื่อง — การสำรองข้อมูลด้วยตนเองที่ไม่เคยแตะ iCloud
- **รีเซ็ตความคืบหน้าของโปรไฟล์** โดยไม่ลบโปรไฟล์ (การตั้งค่าโปรไฟล์ → รีเซ็ต)
- **ลบโปรไฟล์** (จัดการโปรไฟล์ → ปัดหรือแตะเพื่อลบ) — จะนำสำเนา iCloud ออกด้วยเมื่อเปิดการซิงค์ ทำซ้ำต่อโปรไฟล์เพื่อนำออกทั้งหมด

## การเปลี่ยนแปลงนโยบายนี้

หากเราเปลี่ยนนโยบายนี้ เวอร์ชันใหม่จะอยู่ที่ URL เดิมพร้อมวันที่ **อัปเดตล่าสุด** ที่ปรับปรุงแล้ว การเปลี่ยนแปลงที่สำคัญจะถูกระบุไว้ในบันทึกการเผยแพร่ของแอปด้วย

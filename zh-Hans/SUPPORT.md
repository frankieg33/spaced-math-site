# 每日数学 — 支持

常见问题的快速解答，以及如何联系我们。

## 开始使用

1. 打开每日数学。
2. 在欢迎屏幕上，轻点**创建档案**。选择名称、表情符号和颜色。
3. 在档案主页上，轻点某一堆（`+ − × ÷`）或**全部运算**，开始一次学习。说出的答案和输入的答案都可以。

如果你在另一台设备上开启了 iCloud 同步并使用过每日数学，请改为在欢迎屏幕上轻点**从 iCloud 恢复**。iCloud 同步完成后（通常几秒内），你的档案就会出现。如果你在另一台设备上加入了某个家庭，你会看到一张确认卡片，提示你在这里也加入。

## 免费版与完整版

每日数学免费提供一个档案和一种运算（你首次打开应用时选择的那一种）。一次性购买的完整解锁会增加：

- 全部四种运算以及混合练习
- 最多 8 个档案，各自独立的进度
- 在家人设备之间进行家庭共享
- 自定义练习和计时挑战

这是一次性购买，不是订阅，并通过家人共享与你的家人共享。

**如何解锁完整版？**
打开**设置 → 每日数学 完整版 → 解锁完整版**，或轻点任意已锁定的功能。先由大人解出一道简短的乘法题（这是 Apple 对儿童类应用的要求），随后由 App Store 处理付款。

**我已经付过款了。如何恢复？**
在新设备上或重新安装后，用购买时所用的 Apple ID 登录 App Store，然后打开**设置 → 每日数学 完整版 → 恢复购买**。如果是家人购买的，只要你们在同一个家庭群组中，家庭共享就会自动解锁。

**我有促销码或礼品码。**
打开**设置 → 每日数学 完整版 → 兑换代码**并输入。代码被接受后，完整版会立即解锁。

## 语音模式

默认情况下，应用会聆听说出的答案。数字可以自然地说（“二十四”“二 四”“一百四十四”），也可以按数字说。

**语音模式不工作，该怎么办？**

- 确保你已允许访问麦克风和语音识别。iOS 设置 → 每日数学 → 麦克风（开）和语音识别（开）。
- 轻点麦克风区域可放弃当前这次尝试并重试。
- 如果识别一直听错，请切换到屏幕上的键盘：在答案区域轻点**键盘**，或打开档案设置 → 关闭**语音模式**。

**我说的是正确的数字，答案却被判为“错误”。**
语音识别可能会把你说的数字识别成另一个数字。如果经常发生，请暂时切换到键盘，让识别逐渐适应（在同一台设备上通常会随时间改善）。

## 间隔重复

每个档案各自保存独立的进度。你从未见过的卡片是“新卡”，会立即到期。随着你学习：

- 答对会把卡片推向更远的将来。回答越快（默认低于 2.0 秒）间隔越长。
- 答错会立刻把卡片带回轮换：它会在同一次学习中往后 6–12 张重新排队。
- 如果你在本次学习后面又把这张卡片做对了，只计入这次找回的评分，你的“错误”会被忽略。

档案主页上的**统计**栏显示你的连续天数、每天的复习总数，以及已学会／总卡片数。

**复习**卡片（仅在有内容需要复习时显示）是一个有针对性的纠正队列：它会轮流呈现你最近做错的卡片、处于重新学习中的卡片，以及前一天遗留到期的卡片。

## 练习（按档案）

档案主页上的**练习**栏是一种没有排程器的自由练习。卡片池相同，但会忽略到期日期。适合热身，或适合你不想影响 FSRS 排程器的学习。练习难度（简单／中等／困难）可在设置中配置。

## 活动数字

在设置 → 活动数字中，选择你想针对哪些运算数（1–12）进行练习。该选择充当**基数**：选中 `{1, 2}` 时，你会看到涉及 1 或 2 的每一道题，而不仅是 `1+2` / `2+2` / `1+1`。关闭再重新开启某个数字，都不会丢失它的 FSRS 状态。

## iCloud 同步

**如何开启？**
在档案选择器中轻点齿轮图标 → **设置 → 同步 → 使用 iCloud → 开**。确保你已在 iOS 设置中登录 iCloud。

**我的档案没有同步。**

- 确认你在两台设备上都用*同一个* Apple ID 登录了 iCloud。
- 确认两台设备的应用设置中同步都已开启。
- 跨设备的 CloudKit 同步通常很快（几秒，常常通过静默推送几乎瞬时完成），但在网络较弱时可能更久。在每台设备上打开应用，给它 30 秒来稳定下来。

**我切换了 Apple ID，现在档案不见了。**
为了你的安全，应用会在检测到设备上的 iCloud 账户发生变化时自动关闭同步，以免旧数据被悄悄上传到另一个账户。如果你的档案仍在这台设备上，它们安然无恙。如果它们只在之前的 iCloud 账户里，请切换回那个账户并重新开启同步即可恢复。

**我想清除 iCloud 上的副本。**
打开管理档案，逐个删除每个档案。开启 iCloud 同步时，删除也会从家庭区域移除对应的 CloudKit 记录。

## 家庭

每日数学支持把你的整个家庭与其他 Apple ID 共享，这样一个使用爸爸 iPad 和妈妈 iPhone 的孩子，即使这些设备登录的是不同的 iCloud 账户，也能从上次停下的地方接着来。在 v3 中，家庭是共享的单位：所有者设备上的每个档案都属于这个家庭。

**如何创建家庭？**

1. 确保 iCloud 同步已开启（齿轮图标 → 设置 → 同步 → 使用 iCloud → 开）。
2. 打开**设置 → 家庭 → 设置家庭**。
3. 给你的家庭起个名字（例如“张家”）。
4. 选择邀请方式：信息、邮件、拷贝链接或 AirDrop。

**有人接受邀请后会怎样？**

参与者轻点邀请链接，应用会带着确认页打开，他们可在以下之间选择：

- **先导出**：把当前档案保存为一个 JSON 文件（如果改变主意，之后可重新导入）。
- **加入**：用家庭的档案替换其当前档案。**这在参与者的设备上不可撤销** —— 他们加入前的档案会从设备上抹除（但不会从已经持有副本的 iCloud 账户中抹除）。
- **取消**：什么都不做。用户之后可以再次轻点邀请链接。

加入后，参与者会在其选择器中看到家庭名称作为标签，并可查看和编辑家庭中的任意档案。其学习活动会通过 CloudKit 的权限模型写入所有者的 iCloud。

**谁能看到这个家庭？**

只有你明确邀请的 Apple ID。共享仅限受邀者：任何拿到链接的人都可加入，但前提是你把链接发给了他们。你可以随时撤销访问权限（设置 → 家庭 → 解散）。

**数据存储在谁的 iCloud 中？**

所有者的（创建家庭的人）。任何参与者记录的学习活动都会通过 CloudKit 的权限模型写入所有者的 iCloud。参与者的 iCloud 不会保存单独的副本。

**如果所有者解散家庭或退出 iCloud 会怎样？**

所有参与者会在下次刷新时失去实时同步，但家庭档案会**作为他们自己的档案保留在设备上** —— 不会被抹除。如果所有者只是退出了 iCloud，重新登录即可为所有人恢复实时同步。

**参与者能否在不牵涉所有者的情况下离开家庭？**

可以。**设置 → 家庭 → 退出家庭**。所有者的数据不受影响；此后参与者会把家庭的档案作为自己的保留在设备上。

**一个家庭最多可有多少参与者？**

Apple 的 `CKShare` 上限是 100。自然的家庭规模约为 6（与 Apple 家人共享群组的上限一致）。

**持有受管理 Apple ID 的 13 岁以下儿童能加入家庭吗？**

受管理（学校发放）的 Apple ID 不能作为 CKShare 参与者。对于这类家庭，请用*同一个* Apple ID 登录两台设备：按 Apple ID 的同步路径无需邀请也能工作。

## 导出 / 导入（备份）

在**设置 → 备份**中，你可以：

- **将档案导出为文件**：把当前设备上的每个档案保存为一个 JSON 文件。可用作 iCloud 之外的手动备份，或在任何不可撤销的操作（加入家庭、更换设备等）之前使用。
- **从文件导入档案**：恢复之前的导出。**合并**模式会添加缺失的档案，并在导入副本的 `updatedAt` 更新时更新现有档案。**替换**模式会抹除设备上的档案并安装导入的那一组；仅在你当前不在某个家庭中时可用。

## 我想重新开始一个档案而不删除它

打开该档案，轻点齿轮图标，然后选择重置此档案的进度。这会清除间隔重复状态、错误和活动历史，但保留档案的名称、表情符号、颜色和设置。

## 我的设备

- 每日数学可在搭载 iOS 16.0 或更高版本的 iPhone 和 iPad 上运行。
- 除 iCloud 同步（需要登录 iCloud）外，所有功能均可离线使用。

# 隐私政策 — 每日数学

**生效日期：** 2026-05-21
**最近更新：** 2026-06-10

本政策说明 iOS 应用每日数学（“本应用”）如何处理你的信息。它自 v1.0 起适用，包括取代 v2.0 按档案共享的 v3 单一家庭 CloudKit 共享模型。

## 简明摘要

- 我们不运行任何服务器。我们不收集任何个人信息。
- 除非你开启 **iCloud 同步**，否则一切都留在你的设备上；开启后，由 Apple（而非我们）将其存储在你的私人 iCloud 账户中。
- 本应用仅在你于语音模式下作答时使用麦克风。音频由 Apple 的设备端（Siri 级）语音识别处理并立即丢弃；我们绝不存储、上传或分析它。
- 没有任何第三方追踪器、广告或分析 SDK。

## 本应用在你设备上存储的内容

- **档案。** 名称、表情符号、颜色，以及按档案的设置。
- **间隔重复状态。** 对每张卡片（例如 `7 + 8`）：计划的下次复习日期、难度、稳定度和复习次数。
- **活动。** 按档案的每日已完成复习计数（用于连续天数标签和统计屏幕）。
- **错误。** 你最近答错的题目队列，以便本应用重新呈现它们。
- **偏好设置。** 应用范围的开关，例如是否开启 iCloud 同步。

所有这些都写入本应用自己的沙盒化“文稿”文件夹和 `UserDefaults`。本应用无法读取其他应用的数据，其他应用也无法读取本应用的数据。

## 本应用可选地在 iCloud 中存储的内容

iCloud 同步**默认关闭**。当你开启它（齿轮图标 → **设置 → 同步 → 使用 iCloud**）时，本应用会通过 Apple 的 **CloudKit** API 把上述相同数据写入你的私人 iCloud。这是你的 iCloud，只有你在使用同一 Apple ID 登录的设备上才能访问。我们无权访问。

- 具体同步的数据：档案、卡片状态、活动、错误，以及（可选的）家庭共享元数据（你为家庭选择的一个显示名称）。
- iCloud 同步**不包含**任何语音录音的内容，也不包含任何分析。
- 当你用不同的 Apple ID 登录时，本应用会检测到变化并关闭同步，直到你明确重新开启，以免数据被悄悄上传到另一个账户。
- 在**管理档案**中删除某个档案，会将该档案从设备移除，并（当 iCloud 在原始 Apple ID 上可达时）一并从你的 iCloud 副本中移除。

## 家庭共享

在 v3 中，共享发生在**家庭**层级，而非按档案。打开齿轮图标 → **设置 → 家庭 → 设置家庭**，为你的家庭命名，并通过标准的 iOS 共享菜单（信息、邮件或 AirDrop）邀请家人。

- 所有者设备上的所有档案都属于这个家庭。邀请一位家人会把它们一次性全部共享。
- 数据位于**家庭所有者的 iCloud** 中。参与者不会在自己的 iCloud 中保存单独副本；其学习活动会通过 Apple 按区域划分的 CKShare 权限模型直接写入所有者的 CloudKit。
- 加入家庭在**加入的设备上不可撤销**：参与者现有的本地档案会被家庭的档案替换。加入前，本应用会提供**将档案导出为文件**选项，以便你保存当前数据的 JSON 副本，若改变主意可在之后重新导入。
- 如果所有者解散家庭（设置 → 家庭 → 解散）或退出 iCloud，参与者会在下次同步时失去实时访问。家庭档案的本地缓存会成为他们自己的（离开时不会进行不可撤销的抹除）。
- 参与者可随时离开家庭（设置 → 家庭 → 退出）。离开后，家庭的档案会作为他们自己的保留在设备上。
- Apple 的 `CKShare` 上限是每个共享项 100 名参与者。自然的家庭规模约为 6。

## 导出 / 导入（JSON 备份）

在**设置 → 备份**中，你可以：

- **将档案导出为文件**：保存一个包含当前设备上所有档案（无论家庭状态）的 JSON 文件。可用作手动备份或在不可撤销的操作之前使用。
- **从文件导入档案**：恢复之前的导出。**合并**模式会添加缺失的档案，并在导入副本更新时更新现有档案。**替换**模式会抹除本地档案并安装导入的那一组；仅在你不在某个家庭中时可用。

## 麦克风与语音识别

当语音模式开启时，在你作答期间，本应用会：

1. 使用 `AVAudioEngine` 采集实时麦克风音频。
2. 将其流式传输到 `SFSpeechRecognizer`（Apple 的框架）。在较新的 iOS 设备上，识别默认在设备端运行；某些配置可能使用 Apple 的语音识别服务器。
3. 读取转写出的数字并评判答案。
4. 一旦获得最终答案就停止采集。

我们不保留任何音频、转写文本或派生信号。除 Apple 的 `SFSpeechRecognizer` 外，我们不向任何一方传输音频。本应用在首次使用时请求 `NSMicrophoneUsageDescription` 和 `NSSpeechRecognitionUsageDescription`；若任一权限被拒绝，语音模式会自动停用，改用屏幕上的数字键盘。

## 诊断日志

本应用通过 Apple 的标准 `OSLog` 框架输出非结构化的诊断事件（例如“学习结束，正确=12 错误=3”）。这些日志：

- 留在设备上。
- 仅在设备连接到 Mac 时，你才能在 Console.app 中看到。
- 对任何可能识别身份的值（档案 ID、档案名称）使用 Apple 的 `.private` 隐私分类，因此在发布版本中这些值会被隐去。

我们不收集、检索或传输 `OSLog` 内容。

## 我们*不*收集的数据

- 你的姓名、电子邮件或联系方式。
- 你的位置。
- 任何设备标识符（IDFA、IDFV、广告 ID）。
- 超出 Apple 可能通过你的 iOS 设置收集范围的崩溃报告。
- 任何分析或行为遥测。
- 支付或账单信息（购买完全由 Apple 处理）。

## 儿童隐私

每日数学旨在对儿童安全。由于本应用不收集个人信息且不联系第三方，它符合 COPPA 的“无个人信息”定义。我们不展示广告，也没有账户或社交功能。每日数学包含一个一次性的应用内购买（完整版的解锁）；由于本应用为儿童设计，按照 Apple 儿童类目的规则，任何购买之前都会显示成人验证。所有付款和任何代码兑换都由 Apple 通过 App Store 处理。本应用绝不会看到或存储你的支付信息。

## 你的选择

- **关闭语音模式**：随时按档案关闭（设置 → 语音模式），或在 iOS 设置中拒绝麦克风权限以全局关闭。
- **关闭 iCloud 同步**：随时（齿轮图标 → 设置 → 同步 → 使用 iCloud → 关）。关闭不会删除 iCloud 副本；若要移除，请逐个删除每个档案。
- **解散或离开家庭**（设置 → 家庭 → 解散 / 退出）以停止共享。解散会撤销参与者对你家庭的访问；离开会把家庭的档案作为你自己的保留在设备上。
- **将档案导出为文件**（设置 → 备份 → 导出）并保存在本地：一份绝不触及 iCloud 的手动备份。
- **重置档案的进度**而不删除该档案（档案设置 → 重置）。
- **删除档案**（管理档案 → 滑动或轻点删除）：开启同步时也会移除 iCloud 副本。逐个档案重复以移除全部。
